## 贡献土味情话

所有情话均存储在`src/assets/words.json`文件内，请 clone 代码然后提 PR。

P.S. `|`代表换行

## 本地开发

`git clone https://github.com/zerosoul/honeyed-words-generator.git`

`cd honeyed-words-generator && yarn install`

`yarn start`

Let's rock it~
